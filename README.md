# InviteChat.site

A simple payment gateway for Discord premium roles. Users pay $3.99 via Stripe, connect their Discord account, and automatically receive a premium role in your server.

## Features

- Stripe checkout ($3.99 one-time payment)
- Discord OAuth2 login
- Automatic role assignment via Discord bot
- Mobile-responsive Discord-themed UI

## Quick Deploy

Paste this repository URL into [hoston.ai](https://hoston.ai) to deploy instantly.

## Manual Setup

1. Copy `.env.example` to `.env` and fill in your credentials
2. `npm install`
3. `npm start`

See `.env.example` for detailed instructions on obtaining each API key.
